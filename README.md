## FAQ

**Is Celery Executor only for Windows?**  
Yes. This project is described as a Windows desktop executor for Windows 10 and 11 machines.

**Will it still work after Roblox updates?**  
The auto-update engine is intended to help keep the app in step with changes, though Roblox-side updates can still affect compatibility.

**How does it compare with other script executors?**  
Celery Executor puts the emphasis on a compact UI, batch execution, a persistent queue, and a built-in script hub instead of a bulky desktop layout.

**Does using a script executor affect account risk?**  
Any third-party executor may introduce account or policy risk depending on usage and platform rules. Check the terms you depend on before running external tools.

**Are scripts stored locally?**  
The project structure includes local folders for hub content, queue data, updater files, and UI resources, so much of the workflow can be handled directly on the machine.
